Hugo搭建五步曲(Windows)

Hugo搭建方法其实很简单，但在搭建过程笔者因为各种原因踩了不少坑，用时两小时；顺利的话其实用不到二十分钟。如果条件允许，建议在 linux环境上搭建。

一、下载Hugo安装包

  1. 首先在 github-hugo 中找到对应系统的安装包，笔者是：hugo_0.102.3_Windows-64bit.zip，

  2. 进行解压得到hugo.exe文件，将该文件移动到指定目录的bin文件夹，如新创建的 D:\Data\Hugo\bin 目录下

  3. 在环境变量 path中添加该目录，以便系统能够识别hugo命令。

  4. 打开cmd终端，输入hugo version 识别成功

    1
2

    C:\>hugo version
hugo v0.102.3-b76146b129d7caa52417f8e914fc5b9271bf56fc windows/amd64 BuildDate=2022-09-01T10:16:19Z VendorInfo=gohugoio

五、创建你的第一个博客

  1. 先进入站点中的archetypes下修改你的blog模板default.md, 可修改如下
1
2
3
4
5
6

title: "{{ replace .Name "-" " " | title }}"
date: {{ .Date }}
tags: [""]
categories: [""]
description:
draft: true

其中date是文章生成时间，description是文章摘要，draft表示文章是否为草稿（文章写完了修改为 true）。

  1. 回到终端创建你的第一篇文章文档：hugo new posts/first_post.md
1
2

D:\Data\Hugo\Sites\kiteYN>hugo new posts/first_post.md
Content "D:\\Data\\Hugo\\Sites\\kiteYN\\content\\posts\\first_post.md" created

进入first_post.md开始创作^ ^

  1. 最后，构建对应的网页
1

D:\Data\Hugo\Sites\kiteYN>hugo server -D -e production --buildDrafts

就初步完成啦。 –buildDrafts的目的是显示草稿文章，此时再打开md文档进行的修改将实时更新噢
